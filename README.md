## 📋 Описание

**Эльбрус** — профессиональный портативный инструмент для системных администраторов и опытных пользователей, предназначенный для диагностики, восстановления и экстренного ремонта операционной системы Windows. Всё в одном исполняемом файле, не требующем установки.

Инструмент назван в честь самой высокой горы Европы — Эльбруса, символизируя надёжность, мощь и непоколебимость в решении системных проблем.

---

## ✨ Возможности

### 📊 Диспетчер процессов
- Отображение всех запущенных процессов с сортировкой по загрузке CPU
- Индикация процессов с высокой нагрузкой (>50%)
- Принудительное завершение выбранных процессов
- Обновление списка в реальном времени

### 📁 Анализ автозагрузки
- Сканирование критических разделов реестра:
  - **Run** — стандартная автозагрузка
  - **RunOnce** — одноразовая автозагрузка
  - **Winlogon** — контроль входа в систему (обнаружение подмены Shell/Userinit)
  - **AppInit_DLLs** — механизм внедрения DLL
  - **CmdLine** — команды до входа в систему
- Автоматическое обнаружение угроз с подсветкой
- Восстановление Winlogon одним кликом

### 🗂️ Полноценный редактор реестра
- Навигация по всем основным кустам реестра:
  - `HKEY_CLASSES_ROOT`
  - `HKEY_CURRENT_USER`
  - `HKEY_LOCAL_MACHINE`
  - `HKEY_USERS`
  - `HKEY_CURRENT_CONFIG`
- Создание, редактирование и удаление разделов
- Создание, редактирование и удаление параметров
- Поддержка типов данных: `REG_SZ`, `REG_DWORD`, `REG_BINARY`, `REG_EXPAND_SZ`
- Подсветка критических значений Winlogon

### 🔧 Инструменты восстановления
- **Восстановление ассоциаций файлов** (`.exe`, `.lnk`, `.bat`, `.cmd`)
- **Восстановление Winlogon** (Shell = explorer.exe, Userinit = userinit.exe)
- **Очистка временных файлов** (Temp, Prefetch)
- **Удаление подозрительных команд CmdLine**

### 💻 Анализ CmdLine
- Детальное сканирование параметров, запускающихся до входа в систему
- Обнаружение подозрительных команд
- Выявление угроз с использованием сигнатур
- Однокнопочное удаление обнаруженных угроз

## ⚙️ Установка и запуск

1. Загрузите файл `Elbrus.exe`
2. Поместите файл в любое удобное место (рекомендуется отдельная папка)
3. При желании поместите файл `icon.ico` в ту же папку для кастомизации иконки
4. Запустите `Elbrus.exe` от имени администратора
5. При первом запуске разрешите повышение прав (UAC)
6. Используйте вкладки для работы с соответствующими инструментами
